# Menu

トリガー要素をクリックで開き、外側クリックまたはESCで閉じるドロップダウンメニューです。
MenuGroup で項目をグルーピングし、区切り線で視覚的に分離できます。

- Figma URL : [Menu]()

<Canvas of={MenuStories.Default} />

---

## When to Use

- ボタンやアイコンボタンのクリックで複数の操作を提示したいとき
- ページ遷移・アクション実行・設定切り替えなど、異なる種類の操作をひとつのメニューにまとめたいとき
- コンテキストに応じた操作群を省スペースで提供したいとき

## When NOT to Use

- ⚠️ 選択肢が1つしかないとき。ボタンを直接配置してください。
- ⚠️ フォームの選択UIが必要なとき。Select コンポーネントを使ってください。
- ⚠️ 常に表示しておくべきナビゲーションのとき。サイドバーやタブを使ってください。

## Service Context — Speeda AI Agent での使い方

- チャットヘッダーの履歴ボタンから会話履歴一覧を表示
- ユーザーメニュー（プロフィール、設定、ログアウト等）
- コンテンツカードの「もっと見る」ボタンからの操作メニュー

## 構成コンポーネント

| コンポーネント | 役割 |
| --- | --- |
| `Menu` | 開閉状態を管理するルート |
| `MenuTrigger` | トリガー要素（MenuButton, IconButton 等）のラップ |
| `MenuDropdown` | Portal + Positioner + Popup をまとめたドロップダウン本体 |
| `MenuGroup` | アイテムのグルーピング + 区切り線 |
| `SubMenuDropdown` | サブメニュー用ドロップダウン |
| `MenuSubmenuRoot` | サブメニューのルート（Base UI 再エクスポート） |
| `MenuRadioGroup` | SelectMenuItem 用ラジオグループ（Base UI 再エクスポート） |

## アクセシビリティ

- メニューは `ESC` キーで閉じることができます
- メニュー内のアイテムは矢印キーで移動できます
- `modal` がデフォルト `true` のため、メニュー外の操作はブロックされます
