# FilterChipSingleSelectGroup

FilterChip を横並び・折り返しでレイアウトし、選択肢の中から1つだけを選択できるグループコンポーネントです。

- Figma URL : [FilterChipSingleSelectGroup](https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/%F0%9F%90%9D-Base-Component-Speeda-3.1-%E3%83%9F%E3%83%84%E3%83%90%E3%83%81?node-id=5702-6400&t=wAsCASAO2ZF6eRSb-1)

## When to Use

- フィルター条件を排他的に1つだけ選択させたいとき
- ラジオボタンの代替として、よりコンパクトなUIで選択肢を提示したいとき

## When NOT to Use

- ⚠️ 複数の条件を同時に選択させたいとき。代わりに [FilterChipMultiSelectGroup](?path=/docs/components-chip-filterchipmultiselectgroup--docs) を使ってください
- ⚠️ 選択肢が2〜5個の固定的な切り替えのとき。代わりに [SegmentedControl](?path=/docs/components-segmentedcontrol-segmentedcontrol--docs) を検討してください

## Service Context — Speeda AI Agent での使い方

- 検索結果の表示期間（1年、3年、5年など）の切り替え
- データの表示モード（日次、週次、月次など）の選択
