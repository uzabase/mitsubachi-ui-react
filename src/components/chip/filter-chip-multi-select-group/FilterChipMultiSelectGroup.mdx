# FilterChipMultiSelectGroup

FilterChip を横並び・折り返しでレイアウトし、複数同時に選択できるグループコンポーネントです。

- Figma URL : [FilterChipMultiSelectGroup](https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/%F0%9F%90%9D-Base-Component-Speeda-3.1-%E3%83%9F%E3%83%84%E3%83%90%E3%83%81?node-id=7216-14722&t=wAsCASAO2ZF6eRSb-1)

<Canvas of={FilterChipMultiSelectGroupStories.Default} />

## When to Use

- フィルター条件を複数同時に選択させたいとき
- 選択肢をチップ形式で横並びに表示したいとき

## When NOT to Use

- ⚠️ 選択肢から1つだけ選ばせたいとき。代わりに [FilterChipSingleSelectGroup](?path=/docs/components-chip-filterchipsingleselectgroup--docs) を使ってください
- ⚠️ ユーザーの入力内容をタグとして管理したいとき。代わりに [InputChipGroup](?path=/docs/components-chip-inputchipgroup--docs) を使ってください

## Service Context — Speeda AI Agent での使い方

- 検索フィルターで複数の業種やカテゴリを同時に選択する場面
- レポート設定で複数の表示項目を選択する場面
