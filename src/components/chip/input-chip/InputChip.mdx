# InputChip

ユーザーが入力した内容を、要素ごとに整理して表示するコンポーネントです。×ボタンで個別に削除できます。

## When to Use

- ユーザーが入力・選択した項目をタグ形式で一覧表示するとき
- 各項目を個別に削除できるようにしたいとき

## When NOT to Use

- 🚫 単独で配置したいとき。必ず [InputChipGroup](?path=/docs/components-chip-inputchipgroup--docs) 経由で使用してください
- ⚠️ フィルター条件の on/off を切り替えたいとき。代わりに [FilterChip](?path=/docs/components-chip-filterchip--docs) を使ってください
- ⚠️ 読み取り専用のラベルを表示したいとき。代わりに [ReadOnlyTag](?path=/docs/components-tag-readonlytag--docs) を使ってください

## Service Context — Speeda AI Agent での使い方

- 検索条件として入力されたキーワードの一覧表示
- 選択済みのフィルター項目（企業名、タグなど）の表示と削除
