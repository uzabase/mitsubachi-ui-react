# Segment

SegmentedControl 内の個別セグメントです。テキストまたはアイコンで選択肢を表現します。

## When to Use

- SegmentedControl 内で個別の選択肢を構成するとき
- テキストまたはアイコンで選択肢を表現するとき

## When NOT to Use

- 🚫 単独で配置したいとき。必ず [SegmentedControl](?path=/docs/components-segmentedcontrol-segmentedcontrol--docs) 内に配置してください
- ⚠️ 複数選択が必要なとき。[Checkbox](?path=/docs/components-checkbox--docs) グループを検討してください

## Service Context — Speeda AI Agent での使い方

- データの表示形式切り替え（テーブル/チャート）の個別選択肢
- 表示モード（グリッド/リスト）の個別選択肢
