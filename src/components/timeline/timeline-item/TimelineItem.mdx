# TimelineItem

出来事や状態の変化を時系列に沿って表示するコンポーネントの、時系列の個別アイテムです。。<br/>
`emphasized` を指定すると、ドットが有彩色になり重要な出来事を強調できます。

## When to Use

- Timeline コンポーネント内で個別のイベントを表示するとき
- 重要なイベントを `emphasized` で強調したいとき

## When NOT to Use

- 🚫 Timeline の外で単独使用したいとき。必ず [Timeline](?path=/docs/components-timeline-timeline--docs) の子要素として使用してください

## Service Context — Speeda AI Agent での使い方

- 企業の重要イベント（上場、M&A など）を `emphasized` で強調
- 通常のイベント（決算発表、プレスリリースなど）をデフォルトで表示
