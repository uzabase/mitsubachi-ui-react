# InformationDialog

利用規約やお知らせなど、重要な情報を表示するためのモーダルダイアログです。Desktop では small / medium / large の3サイズに対応し、Phone では画面端まで広がりフルスクリーンで表示されます。

## When to Use

- 利用規約やプライバシーポリシーなど、長文のドキュメントを表示するとき
- お知らせや重要な通知を、ユーザーが確認するまで表示し続けたいとき
- コンテンツ量に応じて small / medium / large のサイズを使い分けたいとき

## When NOT to Use

- ⚠️ 操作の確認を求めたいとき。代わりに [ActionDialog](?path=/docs/components-dialog-actiondialog--docs) を使ってください
- ⚠️ フォーム入力を伴うとき。代わりに [FormDialog](?path=/docs/components-dialog-formdialog--docs) を使ってください
- ⚠️ 短い通知を一時的に表示したいとき。代わりに [Snackbar](?path=/docs/components-snackbar--docs) を使ってください

## Service Context — Speeda AI Agent での使い方

- サービス利用規約やプライバシーポリシーの表示
- リリースノートや機能アップデートのお知らせ
- ヘルプドキュメントや操作ガイドの表示

## バリエーション

| バリエーション | 用途 |
| --- | --- |
| `small` | 短いお知らせや簡潔な情報の表示 |
| `medium` | 標準的な情報量のコンテンツ（デフォルト） |
| `large` | 利用規約など長文ドキュメントの表示 |
