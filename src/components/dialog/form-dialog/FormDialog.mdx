# FormDialog

フォーム入力を行うためのモーダルダイアログです。Desktop では medium サイズ固定、Phone では画面端まで広がりフルスクリーンで表示されます。

## When to Use

- ダイアログ内でテキスト入力やセレクトボックスなどのフォーム操作を行うとき
- 新規作成や編集など、複数フィールドへの入力が必要なとき
- メインのページコンテキストを保持したまま、フォーム入力を完了させたいとき

## When NOT to Use

- ⚠️ フォーム入力を伴わない単純な確認のとき。代わりに [ActionDialog](?path=/docs/components-dialog-actiondialog--docs) を使ってください
- ⚠️ 利用規約や長文の情報を読ませたいとき。代わりに [InformationDialog](?path=/docs/components-dialog-informationdialog--docs) を使ってください
- ⚠️ 入力フィールドが1つだけの簡易な入力のとき。インラインの入力フォームで対応できないか検討してください

## Service Context — Speeda AI Agent での使い方

- 新規アイテムの作成フォーム（ステータス追加、メモ作成など）
- 既存アイテムの編集フォーム
- 設定変更やプロフィール編集など、複数フィールドの入力が必要な操作
