# ActionDialog

確認や削除など、ユーザーの意思決定を求めるモーダルダイアログです。Desktop では small サイズ固定、Phone では横に余白を持ちコンパクトに表示されます。

## When to Use

- 操作の実行前にユーザーへ確認を求めるとき（例：「本当に実行しますか？」）
- 削除やリセットなど、取り消しが難しい破壊的操作の最終確認を行うとき
- 短いメッセージとアクションボタンだけで完結する簡潔な確認ダイアログが必要なとき

## When NOT to Use

- ⚠️ フォーム入力を伴う操作のとき。代わりに [FormDialog](?path=/docs/components-dialog-formdialog--docs) を使ってください
- ⚠️ 利用規約や長文の情報を読ませたいとき。代わりに [InformationDialog](?path=/docs/components-dialog-informationdialog--docs) を使ってください
- ⚠️ 単純な通知やフィードバックを表示したいとき。代わりに [Snackbar](?path=/docs/components-snackbar--docs) を使ってください

## Service Context — Speeda AI Agent での使い方

- データの削除やリセット操作の確認ダイアログ
- ステータス変更やワークフロー遷移の最終確認
- ログアウトやセッション終了の確認
