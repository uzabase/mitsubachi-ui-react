---
description: コンポーネントのStorybookドキュメントページ（MDX）を作成する
---

`.claude/skills/create-storybook-docs/SKILL.md` のスキルに記載されているすべてのガイドライン、パターン、ルールに厳密に従って、StorybookのドキュメントページをMDX形式で作成してください。

対象: $ARGUMENTS

- 引数が指定されていない場合は、ユーザーにどのコンポーネントのドキュメントを作成するか確認する
- `all` が指定された場合は、全コンポーネントを対象に一括生成モードで実行する
- コンポーネント名が指定された場合は、そのコンポーネントのみを対象にする
